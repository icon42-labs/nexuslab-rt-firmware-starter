# NexusLab Real-Time Firmware Architecture Starter

Starter code and documentation for the **NexusLab Real-Time Firmware Architecture Masterclass** by **icon42-labs**.

NexusLab teaches portable embedded firmware architecture using **RP2040/RP2350** as the reference platform.

This public repository contains a minimal preview of the course approach. It introduces the typical starting point of many embedded projects: a simple blocking superloop.

The full Masterclass shows how this starting point is transformed step by step into non-blocking firmware, event-driven state machines, interrupt-driven event handling and FreeRTOS-based firmware architecture.

## What is included

- minimal RP2040/RP2350 hardware setup
- one introductory blocking-superloop example
- a high-level architecture preview

## What is covered in the full Masterclass

1. Blocking superloops
2. Non-blocking firmware
3. Event-driven state machines
4. Interrupt-driven event handling
5. FreeRTOS-based firmware architecture

## Reference platform

The starter examples use RP2040/RP2350-based boards as the reference platform.

The architectural concepts are intentionally designed to be portable to other microcontroller platforms, but this starter repository focuses on the RP2040/RP2350 implementation.
